Smart Trolley Using RFID

Overview

The Smart Trolley Using RFID project is an automatic billing system designed for supermarkets and shopping malls. This system uses RFID technology to scan products automatically and display the total bill on an LCD screen. The project helps to reduce billing time, avoid long queues, and improve the shopping experience for customers.


---

Features

Automatic product scanning using RFID

Real-time bill calculation

LCD display for product details and total amount

Easy and user-friendly operation

Reduced waiting time at billing counters

IoT-based smart shopping system

Working Principle

1. Each product contains an RFID tag with unique information.


2. The RFID reader scans the product when it is placed in the trolley.


3. Arduino processes the data and calculates the total bill.


4. Product details and bill amount are displayed on the LCD screen.


5. The final bill can be sent through the IoT module for monitoring and billing purposes.
